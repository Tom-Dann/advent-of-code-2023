### Advent of Code 2023 - Go

## Pre-requisites
 - [Go installed](https://go.dev/)

## Running Solutions

Save the input file as `input.txt` under in the corresponding day's folder (e.g. `day01/input.txt`).
Then go to that folder and run the `main.go` solution

    % cd day01
    % go run .
